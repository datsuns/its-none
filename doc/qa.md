## Q2. 操作対象スプレッドシートの特定方法
- [x] **回答済み**
- **質問**: アプリがどのスプレッドシートを操作するか、どのように決定しますか？
  - A) スプレッドシートIDをアプリにハードコードする（固定ファイル）
  - B) 初回起動時にユーザーがファイルを選択する（Google Drive ピッカー）
  - C) アプリが自動でDrive上に `its-none-data` という名前のファイルを作成・検索する
- **影響範囲**: Drive API の使用方法・初回セットアップフロー
- **回答**: **間接参照方式（設定ファイル経由）** を採用。
  - Drive 上に設定ファイル `its-none-config.json` をファイル名で検索してアクセス
  - 設定ファイルが存在しない場合はアプリ上に警告を表示
  - 設定ファイルのフォーマットは JSON（現状はデータファイルIDのみを格納）
  - 設定ファイル例: `{ "spreadsheet_id": "xxxxxxxxxxxxxxxx" }`
